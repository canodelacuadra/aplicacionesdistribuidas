Nuestra página tendrá que cargar al menos tres librerías para mostrar un gráfico:

  1. API de Google AJAX: Esta API se utiliza para cargar la librería de visualización de Google, las librerías de visualización y realizar otras tareas importantes. Ésta, siempre se debe cargar en primer lugar; las otras dos librerías son cargadas en una sola llamada.
  2. Librería de visualización de Google:Esta librería incluye clases comunes y los métodos utilizados para crear y manejar las visualizaciones, por ejemplo: DataTable para sostener sus datos; Query para la consulta de los proveedores de datos; y errors para ayudarle a atrapar y mostrar errores en la página.
  3. Librerías Individuales de gráficos:Cada tipo de gráfico está representado por una clase de JavaScript. La librería corechart tiene varias clases de tabla común, incluyendo AreaChart , BarChart y PieChart , pero otros apartados requiere que cargue librerías específicas.
